# Parse Dashboard - دليل المستخدم

## معلومات الموقع

**الغرض:** لوحة تحكم متقدمة لإدارة تطبيقات Parse بسهولة وأمان. يمكنك تسجيل الدخول وإدارة المستخدمين والبيانات مباشرة من الواجهة.

**الوصول:** عام للجميع - لا يتطلب تسجيل مسبق

---

## مدعوم بـ Manus

**تقنيات البناء:**
- **الواجهة الأمامية:** React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui
- **الخادم:** Express 4 + tRPC 11 + Node.js
- **قاعدة البيانات:** MySQL مع Drizzle ORM
- **المصادقة:** Manus OAuth متكامل
- **النشر:** بنية تحتية قابلة للتوسع مع شبكة توصيل عالمية (CDN)

---

## استخدام الموقع

### تسجيل الدخول والتسجيل

1. انقر على زر **"تسجيل الدخول"** في الصفحة الرئيسية
2. اختر بين **"تسجيل الدخول"** إذا كان لديك حساب أو **"إنشاء حساب"** للمستخدمين الجدد
3. أدخل اسم المستخدم وكلمة المرور (وبريدك الإلكتروني عند الإنشاء)
4. انقر على **"تسجيل الدخول"** أو **"إنشاء حساب"** للمتابعة

### لوحة التحكم الرئيسية

بعد تسجيل الدخول، ستصل إلى لوحة التحكم حيث يمكنك:

**عرض الإحصائيات:**
- عدد إجمالي المستخدمين في النظام
- عدد الكائنات في الفئة المختارة
- حالة الاتصال بالخادم

**إدارة المستخدمين:**
1. اختر **"المستخدمون"** من الشريط الجانبي
2. سترى جدول بجميع المستخدمين مع أسماء المستخدمين والبريد الإلكتروني
3. انقر على أيقونة **"سلة المهملات"** لحذف أي مستخدم

**إدارة البيانات المخصصة:**
1. أدخل اسم فئة Parse مخصصة في حقل **"فئة مخصصة"**
2. انقر على **"عرض"** لتحميل الكائنات من هذه الفئة
3. استخدم **"إنشاء كائن جديد"** لإضافة بيانات JSON جديدة
4. اعرض جميع الكائنات في الجدول وحذفها حسب الحاجة

**تسجيل الخروج:**
انقر على زر **"تسجيل الخروج"** في أعلى يمين الصفحة

## إدارة الموقع

### لوحة التحكم الإدارية

يمكنك الوصول إلى لوحة التحكم الإدارية من خلال:

- **إعدادات:** عدّل عنوان الموقع والشعار والإعدادات العامة
- **لوحة المعلومات:** اعرض إحصائيات الزيارات والأداء
- **قاعدة البيانات:** أدر جداول قاعدة البيانات مباشرة
- **الأسرار:** أدر مفاتيح API ومتغيرات البيئة بأمان
